\# ATM Interface



\## Description

A Java console-based ATM simulation that allows users to perform banking operations.



\## Features

\- Login Authentication

\- View Balance

\- Deposit Amount

\- Withdraw Amount

\- Transfer Amount

\- Transaction History

\- Exit



\## Technologies Used

\- Java

\- OOP Concepts

\- Collections Framework
